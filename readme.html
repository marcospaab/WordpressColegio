# Ejemplo de Sitio Web de WordPress para un Colegio Este repositorio contiene un ejemplo de un sitio web desarrollado con **WordPress** como parte de un curso de desarrollo web. El sitio está diseñado específicamente para un colegio y cuenta con diversas funcionalidades útiles para su comunidad educativa. --- ## Características del Sitio Web ### 1. **Página Principal** - Presenta una introducción al colegio con información destacada y accesos rápidos a las secciones principales. ### 2. **Noticias y Publicaciones** - Sección dedicada a noticias, anuncios y artículos relacionados con las actividades del colegio. - Compatible con etiquetas y categorías para una mejor organización del contenido. ### 3. **Eventos y Calendario** - Una sección para publicar eventos importantes del colegio. - Calendario interactivo donde se pueden consultar las fechas de actividades clave, reuniones y eventos escolares. ### 4. **Galería de Imágenes** - Galería visual para mostrar fotografías de eventos, actividades y logros del colegio. ### 5. **Área de Contacto** - Formulario de contacto para que los usuarios puedan enviar consultas directamente. - Información básica de contacto (dirección, teléfono y correo electrónico). ### 6. **Redes Sociales** - Enlaces integrados a las redes sociales del colegio para facilitar la interacción con la comunidad. --- ## Tecnologías Utilizadas - **WordPress**: Plataforma CMS principal utilizada para construir el sitio web. - **Temas y Plugins**: Uso de un tema personalizable y plugins para añadir funcionalidad (ejemplo: galería, formulario de contacto y calendario). - **HTML/CSS**: Personalizaciones ligeras para estilos y diseño. --- ## Instalación y Configuración 1. Asegúrate de tener instalado [XAMPP](https://www.apachefriends.org/index.html) en tu equipo. 2. Clona este repositorio en tu entorno local: ```bash git clone https://github.com/marcospaab/WordpressColegio.git ``` 3. Copia los archivos del proyecto en el directorio `htdocs` de tu instalación de XAMPP. Por ejemplo: ``` C:/xampp/htdocs/wordpress-colegio ``` 4. Inicia Apache y MySQL desde el panel de control de XAMPP. 5. Crea la base de datos: - Accede a [XAMPP](http://localhost/). - Crea una nueva base de datos. - Importa el archivo SQL correspondiente al proyecto. 6. Accede al sitio desde tu navegador en [http://localhost/wordpress-colegio](http://localhost/wordpress-colegio). 7. Asegúrate de instalar los plugins necesarios desde el administrador de WordPress. --- ## Créditos Este proyecto fue desarrollado como parte de un curso de desarrollo web. - **Autor:** Marcos Adrián Padín Abal(https://github.com/marcospaab) - **Curso:** Udemy - Mega Curso de Wordpress de Cero a Maestro - **Instructor:** render2web --- ## Licencia Este proyecto es libre de uso. ---
